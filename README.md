# Ds-Project

In this project we have made learning platform using Data Structures in Java and have also created GUI using swing and awt packages of java.

# Data Structures Used:

1) Generic Tree.
2) LinkedList.
3) Hashmap

--------------------------------------------------------------------------------------------------------------
Execution Enviroment :JavaSE-1.8 (jre)

IDE used :- Eclipse
